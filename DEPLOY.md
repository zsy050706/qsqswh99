# 星河寄语 · 生日礼物部署指南

> 给她一份「打开链接就能看」的礼物，不需要她会任何代码。

---

## 她需要做什么？

只需要 **3 步**：

1. 收到你发给她的 **链接** 或 **二维码**
2. 用手机或电脑 **浏览器** 打开（微信里长按链接 → 在浏览器中打开）
3. 点击 **「开启星途」** 按钮，滑动星河探索

不需要安装 App，不需要注册账号。

---

## 推荐方案：Netlify 免费部署（最简单）

**优点**：免费、5 分钟搞定、自动生成 HTTPS 链接、手机电脑都能看。

### 步骤

1. 在电脑上打开 [https://app.netlify.com/drop](https://app.netlify.com/drop)（Netlify Drop 页面）
2. 把整个 `No1` 文件夹 **拖进去**（包含 html、css、js、照片、音频）
3. 等待上传完成，会得到一个类似 `https://random-name.netlify.app` 的网址
4. 点击 Netlify 后台的 **Site settings → Change site name**，改成好记的名字，例如 `our-star-story.netlify.app`

### 做成礼物

- 把网址生成 **二维码**（推荐 [https://cli.im](https://cli.im) 或微信搜「二维码生成」）
- 打印二维码，或写在生日卡片上
- 卡片上可以写：**「扫一下，里面有我们的星河」**

## 备选方案

### 方案 B：GitHub Pages（长期稳定）

1. 注册 GitHub 账号
2. 新建仓库，上传 `No1` 文件夹内所有文件
3. 仓库 Settings → Pages → Source 选 main 分支
4. 获得 `https://你的用户名.github.io/仓库名/love_message.html`

### 方案 C：本地 U 盘 / 微信发文件（无网络也能看）

1. 把整个 `No1` 文件夹复制到 U 盘
2. 让她双击 `love_message.html` 打开
3. **注意**：照片和音频路径要正确，且需在同一文件夹内

### 方案 D：阿里云 OSS（国内访问快）

详见 `readme.txt` 第八节。

## 部署前检查清单

在送给她的之前，你自己先测一遍：

- [ ] 双击 `love_message.html` 能正常打开
- [ ] 点击「开启星途」能进入星河
- [ ] 滑动流畅，星星能点开看文字
- [ ] `click.mp3` 和 `bgm.mp3` 已放入 `No1` 文件夹
- [ ] 照片已放入 `photos/` 并在 `js/data.js` 里填好路径
- [ ] 手机浏览器测试一遍（比电脑更重要）

---

## 音频文件

把这两个文件放在 `No1` 文件夹根目录：

| 文件名 | 用途 |
|--------|------|
| `click.mp3` | 点击按钮时的轻柔音效 |
| `bgm.mp3` | 背景音乐（浪漫、不要太吵） |

浏览器不允许自动播放音乐，所以音乐会在她 **第一次点击按钮** 时才开始，这是正常的。

---

## 修改文字内容

编辑 `js/data.js` 文件，找到 `STAR_DATA` 数组，修改每条记录的：

```javascript
{ photo: 'photos/001.jpg', text: '你的文字...', audio: 'audios/001.mp3' }
```

- `photo`：照片路径（留空 `''` 则显示占位图）
- `text`：星语正文
- `audio`：语音路径（留空则显示占位）

## 常见问题

**Q：微信里打开链接没声音？**  
A：让她点右上角「…」→「在浏览器中打开」，或在 Safari / Chrome 中打开。

**Q：星星点不开？**  
A：滑动时不要误触；停下来再轻触星星图标即可。

**Q：部署后图片不显示？**  
A：检查 `data.js` 里的路径是否正确，文件名区分大小写。

**Q：想改标题或按钮文字？**  
A：编辑 `love_message.html` 里的「星河寄语」「开启星途」等文字即可。
