星河寄语 - 使用说明
====================

【重要】代码已拆分为多个文件，请保持文件夹结构完整：
  love_message.html / css/styles.css / js/data.js / js/app.js
  生日礼物部署指南请阅读 DEPLOY.md

一、如何使用本地文件
--------------------
1. 直接双击 love_message.html 文件即可打开
2. 页面会显示星空背景和古风按钮，点击后进入星途

二、页面结构
------------
一个无限滑动的星河画布，共100颗3D星星：
- 第1-20颗星：相遇
- 第21-40颗星：相识
- 第41-60颗星：相知
- 第61-80颗星：相恋
- 第81-100颗星：相守

三、交互方式
------------
1. 滑动探索：鼠标拖拽或触屏滑动来探索整个星河
2. 点击星星：打开居中模态框，显示照片、文字记录和语音
3. 鼠标悬停：星星会放大并停止旋转，显示编号
4. 惯性滚动：滑动后会有平滑的惯性效果

四、3D星星效果
--------------
- 使用CSS 3D变换创建立体星星
- 星星会自动旋转，展现不同角度
- 悬停时放大并停止旋转
- 带有金色渐变和发光效果

五、视差效果
------------
- 三层背景星空，移动速度不同
- 营造沉浸式的深空探索体验

六、音频文件配置
----------------
需要准备两个音频文件：
1. click.mp3 - 点击按钮时的音效（温和的提示音）
2. bgm.mp3 - 点击后播放的背景音乐（温暖浪漫的音乐）

将这两个音频文件放在与 love_message.html 同一个文件夹中即可。

七、如何添加内容
----------------
打开 js/data.js 文件，找到 STAR_DATA 数组：

每个条目格式如下：
{
    photo: '照片路径',
    text: '你的文字记录',
    audio: '语音文件路径'
}

示例：
{
    photo: 'photos/001.jpg',
    text: '初次相遇，是在那个阳光明媚的午后...',
    audio: 'audios/001.mp3'
}

将你准备好的照片放在 photos/ 文件夹，语音放在 audios/ 文件夹，然后逐个替换占位内容即可。

八、部署到服务器
----------------
方式一：阿里云 OSS（推荐，成本最低）
1. 登录阿里云控制台，打开 OSS 服务
2. 创建一个 Bucket，设置为公共读
3. 将 love_message.html、click.mp3、bgm.mp3 以及 photos/ 和 audios/ 文件夹上传到 Bucket 中
4. 获取文件的公共访问 URL，即可在线访问

方式二：阿里云 ECS 服务器
1. 购买阿里云 ECS 服务器（推荐轻量应用服务器，配置足够）
2. 安装 Nginx 或 Apache 服务
3. 将文件上传到 /usr/share/nginx/html/ 目录
4. 配置服务器安全组开放 80 端口
5. 通过服务器公网 IP 访问

方式三：免费静态托管（推荐）
1. GitHub Pages - 将文件上传到 GitHub 仓库，开启 Pages 功能
2. Netlify/Vercel - 直接拖拽上传文件即可部署

九、关于在线访问
----------------
部署完成后，对方只需要：
1. 打开浏览器
2. 输入你提供的网址
3. 点击页面上的按钮即可查看

不需要安装任何软件，不需要代码，直接点开网页就行。

十、关于浏览器自动播放
----------------------
浏览器出于安全考虑，禁止网页自动播放音频。
解决方案：点击按钮时开始播放音频，这是浏览器允许的用户交互触发。
当前代码已实现：点击按钮时同时播放点击音效和背景音乐。

十一、是否需要后端
-------------------
不需要后端！这是一个纯静态的HTML页面，所有内容都在前端实现。
你只需要将HTML文件和相关资源（图片、音频）上传到服务器即可。

十二、页面特性
--------------
- 3D星星：CSS 3D变换创建的立体星星，自动旋转
- 无限滑动：拖拽探索整个星河画布，支持惯性滚动
- 视差效果：三层背景星空，营造深空感
- 古风配色：深蓝夜空背景，金色点缀
- 闪烁动画：星星逐个闪烁出现
- 点击星星打开模态框：显示照片、文字、语音
- 古风字体：使用思源宋体，古典优雅
